# Tic-Tac-Toe

### `Description:`
The app is based on the Vite React TypesScript and is the basic version for two players.
Player vs CPU should be set up.

### `What used:`
1. TypeScript
2. RTK - Redux Toolkit - possible to build with context
3. MUI
